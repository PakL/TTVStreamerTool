sendmessage

This event is fired when a message is being sent by the user. If a message is not prevented a chatmessage event will follow. Use this event only if you're trying to prevent certain messages from being sent.
Parameters:
Name Type Description
event Object An preventable sendmessage event object.
prevent Boolean Setting this boolean prevents the message from being sent
messageobj Object An object with information about the message that is set to be sent
messageobj.channel String The channel that the message is being sent to
messageobj.message String The message that is being sent (if not prevented). Changing this will *not* change the message that is being sent
messageobj.tags String An emulated tags segment, that the message might have
messageobj.user String An user object of the sender
