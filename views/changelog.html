0.3.9 patch

  • Helix hybrid mode: The new twitch api named Helix is now used where I was able to apply it. For anything else the old twitch api is used.
  • Offline follows will be grayscaled
  • Added an option to hide notification/action stream
  • You can now open a twitch stream player by clicking on the bar above the chat when channel is online
  • Two basic chat filters are now available to help you keep track of important stuff

0.3.7 patch

  • Updated electron to version 2.0.0

With the Electron 2.0.0 came a feature to hint to some possible security concerns. Complying to these some changes had to be made, making current addons incompatible. So I had to add a check to which version the addon is compatible and not load it if not. All official addons were have an update you can install to make them work again.
